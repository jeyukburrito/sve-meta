## 1. 게임 개요

- 2인 대전 카드게임
- **승리 조건**: 상대 리더 체력을 0 이하로 만듦 / 상대 덱 0장일 때 드로우 강제
- **패배 조건**: 자신의 리더 체력 0 이하 / 덱 0장에서 드로우 / 투항 선언
- **무승부**: 양측 동시 패배
- **대원칙**: 카드 텍스트는 룰에 우선한다. 실행 불가능한 지시는 무시한다. 금지 효과는 항상 우선.

## 2. 카드 정보

### 카드 종류
| 종류 | 영문 표기 | 특수 종류 |
|------|----------|----------|
| 리더 | Leader | - |
| 팔로워 | Follower | EVOLVE, ADVANCE |
| 아뮬렛 | Amulet | - |
| 스펠 | Spell | - |

### 주요 정보
- **카드명**: 고유 명칭. 『명칭』 표기는 카드명 완전일치 또는 부분일치
- **클래스**: 뉴트럴/엘프/로열/위치/드래곤/나이트메어/비숍
- **타입**: 룰상 의미 없음. 다른 카드에서 참조하는 특징어
- **코스트**: 플레이에 필요한 PP 수
- **공격력**: 공격 시 부여하는 데미지 기준값
- **체력**: 파괴에 필요한 데미지 수. 리더 초기값 20
- **텍스트**: 능력·효과 기술. () 안의 주석문은 게임에 영향 없음

## 3. 플레이어 정보

### PP/EP/SEP
| 값 | 하한 | 상한 | 용도 |
|----|------|------|------|
| PP 최대값 | 0 | 10 | 턴 시작 시 PP를 이 값으로 설정 |
| PP | 0 | PP 최대값 | 카드 플레이·진화의 코스트 |
| EP | 0 | 없음 | 진화 코스트의 PP 1점을 EP 1점으로 대체 가능 |
| SEP | 0 | 없음 | 초진화에 소비 (초기값 1) |

### 초기값
- 선공 EP: 0 / 후공 EP: 3
- SEP: 각 1
- 리더 체력: 각 20
- PP·PP 최대값: 각 0

### 경과 턴 수
- 각 플레이어가 턴 플레이어로서 스타트 페이즈를 개시한 횟수

## 5. 턴 구조

```
┌─ 스타트 페이즈 ──────────────────────────┐
│ 1. PP 최대값 +1 (10 미만이면)               │
│ 2. PP = PP 최대값으로 회복                   │
│ 3. 자기 필드 카드 전부 스탠드                │
│ 4. 1장 드로우 (선공 1턴째는 불가)            │
│ 5. 체크 타이밍                               │
└──────────────────────────────────────────────┘
         ↓
┌─ 메인 페이즈 ────────────────────────────┐
│ 아래를 반복 선택:                             │
│  • 카드 플레이 (핸드/EX 에리어에서)          │
│  • 기동 능력 플레이                           │
│  • 팔로워로 공격                              │
│  • 메인 페이즈 종료 → 엔드 페이즈로         │
│ ※ 각 행동 후 체크 타이밍 발생                │
└──────────────────────────────────────────────┘
         ↓
┌─ 엔드 페이즈 ────────────────────────────┐
│ 1. 엔드 페이즈 유발                          │
│ 2. 체크 타이밍                                │
│ 3. 수호 소지 팔로워를 원하는 수만큼 액트     │
│ 4. 체크 타이밍                                │
│ 5. 비턴 플레이어가 퀵 사용 가능              │
│ 6. 핸드 상한 초과 → 버림                     │
│ 7. 「턴 종료까지」 효과 제거                 │
│ 8. 턴 종료 → 상대 턴으로                     │
└──────────────────────────────────────────────┘
```

## 6. 메인 페이즈 상세

### 카드 플레이 절차
1. 카드를 지정·공개하고 해결 영역으로 이동
2. 전제 선택지 결정 (초이스 등)
3. 목표(「선택」대상) 선택
4. 값의 할당 결정
5. 코스트 결정·지불
6. 필드 상한 확인 (팔로워/아뮬렛)
7. 「플레이되었다」 사건 발생
8. 해결 (필드에 놓기 or 효과 실행)
9. 잔존 카드 → 묘지 / 능력 → 제거

### 공격 절차
1. **공격 팔로워 선택**: 자신의 스탠드 상태 팔로워
   - 조건: 턴 처음부터 연속 마스터 or 이번 턴 진화 완료
2. **공격 목표 선택**:
   - 상대 액트 상태 팔로워 (항상 가능)
   - 상대 리더 (턴 처음부터 연속 마스터인 경우만)
   - 수호 소지 액트 팔로워가 있으면 우선 선택
3. 공격 팔로워를 액트
4. 「공격했다」 사건 발생 (목표가 팔로워면 교전 상태)
5. 체크 타이밍
6. 비턴 플레이어의 퀵 사용 기회
7. **데미지 교환**: 공격 팔로워 공격력 → 목표 / 목표 공격력 → 공격 팔로워 (동시)
8. 체크 타이밍 → 공격 종료

## 7. 진화 시스템

### 통상 진화
- **기동 능력**으로 실행 (1턴 1회 제한)
- **코스트**: 에볼브 덱에서 적정 카드 1장 공개 + PP (PP 1점을 EP 1점으로 대체 가능)
- **처리**: 에볼브 카드를 진화 영역에 놓고 필드 카드에 관련부여
- **효과**: 코스트 이외의 정보가 진화 후 카드 정보로 변경. 공격력/체력 차분은 계승
- 진화 전 능력은 상실, 진화 후 능력을 획득
- 스탠드/액트 상태는 불변

### 초진화
- **조건**: 경과 턴 수가 선공 7 이상 / 후공 6 이상
- **추가 코스트**: SEP 1점
- **추가 효과**: 공격력+1/체력+1, 「초진화했다」 사건 발생

## 8. 능력 분류

| 종류 | 표기 예시 | 설명 |
|------|----------|------|
| 기동 능력 | ◆(코스트):(효과) | 플레이어가 능동적으로 사용 |
| 자동 능력 | (조건)때, (효과) | 유발 조건으로 자동 플레이 (강제) |
| 영속 능력 | (위 이외) | 상시 효과 발생 |
| 스펠 능력 | 스펠의 텍스트 | 스펠 고유 |

### 효과 분류
- **단발 효과**: 1회 실행 후 종료
- **계속 효과**: 일정 기간 유효 (「이 게임 중」 포함)
- **치환 효과**: 사건 A 대신 사건 B를 실행 (「~하는 대신」 표기)

## 9. 체크 타이밍

1. 룰 처리를 전부 동시 실행 (반복)
2. 턴 플레이어의 대기 자동 능력 1개 플레이·해결 → 1로 복귀
3. 비턴 플레이어의 대기 자동 능력 1개 플레이·해결 → 1로 복귀
4. 종료

### 주요 룰 처리
- 리더 체력 0 이하 → 패배
- 덱 0장 드로우 → 패배
- 팔로워 체력 0 이하 → 파괴 (묘지로)
- 필살과 교전한 팔로워 → 파괴
- 필드 매수 초과 → 초과분을 묘지로

## 11. 덱 구축 룰

- **리더 카드**: 1장
- **메인 덱**: 40~50장 (리더/에볼브/어드밴스/토큰 불가)
- **에볼브 덱**: 0~10장 (에볼브/어드밴스만)
- **동명 제한**: 메인 3장 + 에볼브 3장 = 총 6장까지
- **클래스 기준**: 전 카드가 리더와 동일 클래스 또는 뉴트럴
- **타이틀 기준**: 전 카드가 동일 타이틀 (리더 포함)

## 13. 타이틀별 키워드

### 13.1 우마무스메 프리티 더비

**식사 (특정표기)**
- 「N회 식사한다」: 카드에 관련부여된 출주 영역 카드가 없을 경우, 에볼브 덱의 『にんじん(당근)』 N장을 출주 영역에 놓고 관련부여
- 이미 관련부여 카드가 출주 영역에 있으면 아무것도 하지 않음 (코스트 불가)
- 필드를 떠나면 출주 영역과의 관련부여 상실

**식사 능력 (키워드)**
- 식사 처리를 포함하는 기동 능력. PP 1점을 EP 1점으로 대체 가능. **진화 상당 능력**

**출주 (특정표기)**
- 「N회 출주한다」: 출주 영역에 관련부여 카드가 있는 한 【돌진】 보유. 「출주했다」 사건 N회 발생

**출주 시 (키워드)**
- 카드가 출주했을 때 유발하는 자동 능력

### 13.2 아이돌마스터 신데렐라 걸즈

**마법의 아이템**
- 이 타이틀 기준 덱인 경우, 선공 결정 직전에 EX 에리어에 『魔法のアイテム(마법의 아이템)』 토큰 5개 배치

**레슨 N (특정표기)**
- EX 에리어의 『魔法のアイテム』을 N장 소멸시킴

### 13.3 카드파이트!! 뱅가드

**트리거 아이콘**
- 카드 우상단의 아이콘. 4종: 크리티컬/드로우/스탠드/힐

**덱 구축 추가 조건**
- 【스타트 아뮬렛】 소지 카드 1장 이상 필수
- 트리거 소지 카드의 클래스는 리더와 일치 필수

**게임 시작 절차 변경**
- 덱 배치 전에 【스타트 아뮬렛】 소지 카드 1장을 뒷면으로 필드에 놓음
- 핸드 교환 후 앞면으로 전환

**스타트 아뮬렛 (키워드)**
- 덱 구축 제한 영속 능력. 이 카드와 다른 카드명의 【스타트 아뮬렛】은 투입 불가

**드라이브 체크 (특정표기)**
- 덱 위 1장 → 트리거 영역. 트리거 아이콘이 있으면 대응 처리를 임의 실행:
  - 크리티컬: 자기 팔로워 1체 공격력 +2
  - 드로우: 1장 드로우
  - 스탠드: 자기 팔로워 1체 스탠드 (해당 턴 상대 리더 공격 불가)
  - 힐: 자기 리더 체력 +3
- 트리거 처리 실행 → 카드 묘지 / 트리거 없음 or 미실행 → 덱 아래

**싱글 드라이브 / 트윈 드라이브 (키워드)**
- 【공격 시】 드라이브 체크 1회 / 2회 연속 실행

**드라이브 (키워드)**
- 팔로워가 얻는 영속 능력. 취득 시 【싱글 드라이브】와 【돌진】도 동시 취득
- 한번 드라이브를 가진 적이 있으면 재취득 불가

**드라이브 획득 시 (키워드)**
- 드라이브를 얻었을 때 유발하는 자동 능력

**빙의/라이드 (키워드)**
- 팔로워에게 드라이브를 갖게 하는 기동 능력
- 코스트: 에볼브 덱의 『ドライブポイント(드라이브 포인트)』 1장을 드라이브 영역에 놓고 관련부여 + PP (PP 1점을 EP 1점으로 대체 가능)
- 이 게임 중 1회만. **진화 상당 능력**
- 필드를 떠나면 드라이브 영역과의 관련부여 상실

### 13.4 프린세스 커넥트! Re:Dive

**유니온 버스트 (키워드)**
- 「유니온 버스트 (능력)」 형식. 「UB」 표기 있음
- **유효 조건**: 타이틀 기준으로 「프린세스 커넥트! Re:Dive」 덱인 경우에만 유효. 클래스 기준이나 다른 타이틀 기준에서는 상시 무효
- 「발동했다」 = 적정하게 플레이되어 해결됨
- 「원래 코스트를 지불하지 않고 발동한다」 = 코스트 0으로 플레이. X 포함 시 X=0

**이퀴프먼트 (종류) / 장비 (특정표기)**
- 토큰으로 생성. 이퀴프먼트 영역에서만 존재 가능
- 「팔로워는 명칭을 장비한다」 = 토큰 생성 → 관련부여 → 「장비했다」 유발
- 팔로워는 복수 장비 가능. 필드를 떠나면 관련부여 상실

## 14. 특정 표기 퀵 레퍼런스

- **파괴하다**: 필드 → 오너 묘지
- **소멸하다**: 임의 영역 → 오너 소멸 영역
- **찾다**: 덱 확인 → 카드 취득 → 셔플
- **뽑다(드로우)**: 덱 위 → 핸드 (덱 0장이면 패배)
- **버리다**: 핸드 → 자기 묘지
- **변신하다**: 소멸 → 동수의 토큰 생성
- **진화하다**: 에볼브 카드를 진화 영역에 관련부여
- **데미지**: 체력을 수치만큼 감소 (음수값 가능)
- **회복하다(PP)**: PP에 값을 가산 (최대값 초과 불가)
- **빼앗다/넘기다**: 마스터 변경. 필드→필드 이동. 오너는 불변
- **초이스하다**: 선택지에서 지정 수 선택 후 실행 (플레이 시 결정)
- **X**: 텍스트 내에서 결정 or 플레이 시 플레이어가 결정
- **~당(につき)**: 단위 수 × 수치의 합계를 1회로 실행 (나머지 버림)
- **절반**: 홀수면 올림
- **상자 채움(箱詰め)**: 능력 상실 + 스타트 페이즈의 룰 스탠드 불가 (효과에 의한 스탠드는 가능)
- **조종하다(操縦)**: 공격력·체력을 가진 아뮬렛을 해당 턴 중 팔로워로 만듦
